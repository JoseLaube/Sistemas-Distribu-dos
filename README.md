# Projetos de Sistemas Distribuídos - Ciência da Computação

Este repositório contém as implementações práticas desenvolvidas para a disciplina de **Sistemas Distribuídos** do curso Ciência da Computação da Universidade do Estado de Santa Catarina (UDESC). Os projetos exploram diferentes formas de comunicação entre processos e arquiteturas cliente-servidor.

### Estrutura do Repositório

| Nome da Pasta | Descrição do Projeto | Tecnologia |
| :--- | :--- | :--- |
| `rmi_porto` | Sistema de gerenciamento de um porto (docas, navios e cargas) utilizando a invocação de métodos remotos. Permite operações centralizadas de cadastro e embarque. | **Java RMI** |
| `multicast` | Implementação de comunicação em grupo, onde uma mensagem enviada por um nó é recebida por todos os participantes do grupo multicast simultaneamente. | **Python Multicast / UDP** |
